# Tripadvisor Clone Homepage

A responsive, modern homepage inspired by Tripadvisor — built with **HTML** and **CSS**.  
This project demonstrates attractive UI design, scrollable sections, and mobile‑friendly navigation.

---

## ✨ Features
- 🌿 **Sticky header** with navigation links
- 🌅 **Hero section** with gradient overlay and search bar
- 🏙️ **Top Destinations** cards (Paris, New York, Tokyo, Rome)
- 🏨 **Plan Your Stay** section (Luxury Hotels, Beach Resorts, City Tours)
- ✈️ **Popular Experiences** section (Beach Adventures, Mountain Treks, Wildlife Safaris, Luxury Cruises)
- 💬 **Traveler Reviews** styled like testimonials
- 📱 **Responsive design** with media queries
- 🍔 **Hamburger menu** for mobile navigation
- 🔖 **Favicon/logo** displayed in the browser tab
